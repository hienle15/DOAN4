# laptopshop
# git init	Khởi tạo repository cục bộ.
# git add .	Thêm tất cả file vào staging area.
# git commit -m "msg"	Commit các thay đổi với thông điệp msg.
# git remote add origin <url>	Kết nối repository cục bộ với GitHub.
# git push -u origin main	Đẩy nhánh main lên GitHub.
# git pull origin main	Đồng bộ thay đổi từ GitHub về máy cục bộ.
# git status	Kiểm tra trạng thái các file trong dự án.
# git log	Xem lịch sử commit.
